## Get started

Install 1host

```bash
npm install -g @1hostjs/hosting
```

Add 1host by adding a new script in package.json

```json
"scripts": {
  "start": "1host"
}
```

Create a 1host.config.js

```js
module.exports = {
  modules: [],
  port: 3000,
};
```
